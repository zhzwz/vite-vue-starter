# Vite Vue Starter

## Usage

- npm

```shell
npx degit zhzwz/vite-vue-starter ProjectName
```

- pnpm

```shell
mkdir ProjectName
cd ProjectName
pnpm dlx degit zhzwz/vite-vue-starter . --force
```
