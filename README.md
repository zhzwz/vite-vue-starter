# Vite Vue Starter

My starter template for Vue & Vite app.

- Vue 3
- Vite 5

## Usage

```sh
pnpm dlx degit zhzwz/vite-vue-starter . --force
corepack enable pnpm
pnpm install
pnpm dev
```
